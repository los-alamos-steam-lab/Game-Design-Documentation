Top Down RPG: Making it Your Own¶

So now that you know how everything works, how do you go about making it your own? This set of lessons will walk you through how to create your own map areas, enemies, and weapons.

This is a resource you’ll probably want to come back to each time you create a new element, as it tracks all of the details that need to be set and reset.
