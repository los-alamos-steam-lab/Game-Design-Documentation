GIMP¶

Summary¶

GIMP is the Gnu Image Manipulation Program and has been around for a very long time. It creates bitmap art and is great for tasks like editing tilesheets.

Getting Started¶

GIMP runs on MacOS, Linux, and Windows. Download from https://www.gimp.org
